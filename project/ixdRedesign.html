Ixd redesign

This project was a short two day sprint with the task of re-designing and build a new visual concept for the Chalmers interaction design programme website.

Client

Chalmers University

Project role

UX Design / UI Design

Date

October, 2015

Background

The task were based on re-designing the interaction design website which accommodates information for vastly different users. The practice was also to facilitate development of the website by researching frameworks and standards. Further, a style guideline was provided to obstruct the design challenge.

Process

Different Chalmers websites were studied to inform the concept of necessary information, functions, and applicable element hierarchy for the new design. The use of a similar structure facilitates the users recognition for the new website. The new concept focus on accommodating current students, and teachers as well as people not yet enrolled who are interested to read more about program and courses. Thus, interviews were with current students as well as outsiders to pinpoint problems with the current site and possible improvements. The collection of information, research about web development standards together with the provided style guide resulted in a minimal and modern look and feel which aims to covey and promote a feeling about the program and discoverability of information.
